# Plan: 보드 옵션 (Board Options)

## 메타 정보
- **Feature**: board-options
- **작성일**: 2026-03-20
- **Phase**: Plan
- **상태**: In Progress
- **기반**: 2048-game (완료, Match Rate 93%)

---

## 1. 목표 (Goal)

기존 2048 게임에 두 가지 기능을 추가한다:
1. **보드 크기 선택** — 4×4 / 5×5 중 선택하여 게임 시작
2. **Undo** — 직전 이동 1단계 되돌리기

---

## 2. 배경 (Background)

- 현재 게임은 4×4 고정. 보드 크기 선택지를 주면 플레이 다양성 증가
- 5×5는 공간이 넓어 비교적 난이도가 낮고, 캐주얼 플레이어에게 적합
- Undo는 실수 구제 기능으로 UX 만족도를 크게 높임
- 두 기능 모두 기존 구조(순수 HTML/CSS/JS)에서 최소 변경으로 구현 가능

---

## 3. 요구사항 (Requirements)

### 3.1 핵심 기능 (Must Have)

| ID | 기능 | 설명 |
|----|------|------|
| F-11 | 보드 크기 선택 | 게임 시작 전/중 4×4 또는 5×5 선택 버튼 |
| F-12 | 크기별 게임 초기화 | 선택한 크기에 맞게 보드, CSS 그리드 동적 변경 |
| F-13 | Undo (1단계) | 직전 이동 이전 상태(보드, 점수)로 복원 |
| F-14 | Undo 버튼 UI | 헤더에 Undo 버튼 표시, 사용 불가 시 비활성화 |

### 3.2 동작 규칙

| 규칙 | 내용 |
|------|------|
| 크기 변경 시 | 게임 재시작 (현재 진행 초기화) |
| Undo 횟수 | 1회만 (이전 이전 상태는 저장 안 함) |
| Undo 후 | history 비워서 연속 Undo 불가 |
| 게임 오버/승리 후 Undo | 허용 (게임 계속 가능) |
| 새 게임 시 Undo history | 초기화 |
| 베스트 스코어 | 4×4 / 5×5 별도 저장 |

### 3.3 미포함 (Out of Scope)

- 3×3, 6×6 등 다른 보드 크기
- 다단계 Undo (여러 번 되돌리기)
- Undo history localStorage 저장

---

## 4. 기술 영향 범위

| 파일 | 변경 내용 |
|------|----------|
| `index.html` | 크기 선택 버튼(4×4/5×5), Undo 버튼 추가 |
| `style.css` | 5×5 그리드 대응 CSS 변수, Undo 버튼 스타일, 비활성화 스타일 |
| `game.js` | `BOARD_SIZE` 변수화, `state.history` 추가, `undo()` 함수, `bestScores` 분리 |

---

## 5. 성공 기준 (Success Criteria)

- [ ] 4×4 ↔ 5×5 전환 시 보드가 올바르게 재렌더링됨
- [ ] 5×5에서 이동/병합 로직이 정확히 동작
- [ ] Undo 버튼 클릭 시 이전 보드/점수로 복원
- [ ] Undo 사용 후 버튼 비활성화
- [ ] 게임 오버 후 Undo로 게임 재개 가능
- [ ] 4×4 / 5×5 베스트 스코어 별도 저장/표시
