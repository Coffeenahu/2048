# Plan: 2048 게임

## 메타 정보
- **Feature**: 2048-game
- **작성일**: 2026-03-20
- **Phase**: Plan
- **상태**: In Progress

---

## 1. 목표 (Goal)

클래식 2048 게임을 순수 HTML/CSS/JavaScript로 구현한다.
별도 빌드 도구 없이 브라우저에서 바로 실행되며, Vercel에 즉시 배포 가능한 형태로 만든다.

---

## 2. 배경 (Background)

- 2048은 4×4 그리드에서 같은 숫자 타일을 합쳐 2048을 만드는 슬라이딩 퍼즐 게임
- 심플한 규칙으로 누구나 즐길 수 있으며, 게임 로직 구현 연습에 적합
- 현재 레포지토리는 Vercel 배포 환경이 세팅되어 있음

---

## 3. 요구사항 (Requirements)

### 3.1 핵심 기능 (Must Have)

| ID | 기능 | 설명 |
|----|------|------|
| F-01 | 4×4 게임 보드 | 16칸 그리드 렌더링 |
| F-02 | 타일 이동 | 키보드 방향키(↑↓←→)로 상하좌우 이동 |
| F-03 | 타일 병합 | 같은 숫자 타일이 충돌 시 합산 |
| F-04 | 새 타일 생성 | 이동 후 빈 칸에 2 또는 4 랜덤 생성 |
| F-05 | 점수 계산 | 병합 시 합산된 값을 점수에 추가 |
| F-06 | 게임 오버 감지 | 이동 불가 시 게임 종료 표시 |
| F-07 | 게임 재시작 | "New Game" 버튼으로 초기화 |

### 3.2 추가 기능 (Should Have)

| ID | 기능 | 설명 |
|----|------|------|
| F-08 | 타일 애니메이션 | 이동/생성/병합 시 CSS 트랜지션 |
| F-09 | 최고 점수 저장 | localStorage로 베스트 스코어 유지 |
| F-10 | 모바일 스와이프 | 터치 제스처로 이동 지원 |

### 3.3 미포함 (Out of Scope)

- 실행 취소(Undo) 기능
- 5×5 이상 보드 크기 옵션
- 서버 기반 리더보드
- 다크모드 토글

## 4. 기술 스택 (Tech Stack)

| 항목 | 선택 | 이유 |
|------|------|------|
| 마크업 | HTML5 | 빌드 불필요, 간단한 구조 |

| 배포 | Vercel | 기존 설정 활용 |

## 6. 게임 로직 개요

### 이동 알고리즘
1. 이동 방향에 따라 각 행/열을 슬라이드
2. 빈 칸 제거 → 인접 동일 숫자 병합 → 다시 빈 칸 제거
3. 병합된 타일은 같은 이동에서 재병합 불가
4. 변화가 있었을 때만 새 타일 생성

### 타일 숫자별 색상
- 2: 연베이지, 4: 베이지, 8: 주황, 16: 진주황
- 32~512: 붉은 계열, 1024~2048: 노란 계열

## 7. 성공 기준 (Success Criteria)

- [ ] 4방향 이동이 정확히 동작
- [ ] 병합 로직 오류 없음 (같은 이동에 이중 병합 없음)
- [ ] 게임 오버/클리어 정확히 감지
- [ ] 모바일 브라우저에서 터치로 플레이 가능
- [ ] 새로고침 후에도 최고 점수 유지
- [ ] Vercel 배포 후 정상 동작
