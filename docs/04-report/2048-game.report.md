### Design (설계)

**문서**: `docs/02-design/features/2048-game.design.md`

**주요 설계 결정**:

1. **UI 레이아웃**
   - Header: 제목, 현재 점수, 최고 점수, New Game 버튼
   - Board: 4×4 CSS Grid (16개 셀)
   - Overlay: Game Over/Win 메시지

2. **기술 결정**
   - **렌더링**: 전체 재렌더링 방식 (성능 vs 단순성 → 단순성 우선)
   - **타일 위치**: CSS 변수 `--r`, `--c`로 계산 (직관적, 부드러운 전환)
   - **상태 저장**: localStorage (베스트 스코어만)
   - **파일 분리**: 단일 3파일 (빌드 도구 불필요)

3. **핵심 알고리즘**
   - **slide()**: 한 줄 슬라이드 + 병합
     - 0 제거 → 인접 동일값 병합 → 우측 0 패딩
     - 병합된 타일 재병합 방지 (`merged` 배열)
   - **move()**: 방향별 이동 처리 (left/right/up/down)

4. **애니메이션**
   - pop-in: 신규 타일 0 → 100%
   - merge: 병합 타일 1 → 1.2 → 1
