- `state.won` 선언만 있고 2048 타일 생성 시 승리 표시 로직 없음
- **수정**: `move()` 내 2048 도달 확인 → `showOverlay('You Win!')` 호출

### Gap-02: `.tile-merged` 클래스 미적용 (Medium)
- CSS에 `@keyframes merge-pop` + `.tile-merged` 정의됨
- JS `render()`에서 병합된 타일에 클래스 미적용
- **수정**: 병합 발생 위치 추적 후 해당 타일에 클래스 추가

### Gap-03: Design 문서와 구현 간 사소한 불일치 (Low)
- `--cell-bg` 값 변경, 애니메이션 이름 변경, 함수 구조 단순화
- 구현이 더 완성도 높으므로 Design 문서 업데이트 권장
